# OCR Keeper

ダークモードUIのブラウザOCRツールです。複数画像から文字列抽出し、固定枠で保存・編集・コピーできます。

## 仕様
- 最大20ページ / 1ページ10枠
- 枠は小さく表示し、1ページ10枠を一覧しやすいレイアウト
- 写真は追加選択でキュー末尾に追記（上限200枚）
- 200枚を超える選択は自動的に取り込まない
- 検索条件はトグルで開閉
- 検索条件は「先頭一致 / 末尾一致」のどちらかを選択、文字列は複数文字対応
- OCR進行率バーで読み込み進捗を表示
- OCR失敗時は「読み取れませんでした」で枠消費
- コピー履歴テキストは非表示、コピー済み文字列を色変化
