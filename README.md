# OCR Keeper

ダークモードUIのブラウザOCRツールです。複数画像から文字列抽出し、固定枠で保存・編集・コピーできます。

## 仕様
- 最大20ページ / 1ページ10枠
- 写真は追加選択でキュー末尾に追記（上限200枚）
- 選択中枚数と抽出可能件数を表示し、抽出可能件数に応じて抽出ボタン文言を更新
- 抽出完了後は選択キューをクリアし、抽出ボタンを通常表示へ戻す
- 検索条件は常時表示（先頭一致 / 末尾一致 + 複数文字条件）
- OCR進行率バーで読み込み進捗を表示
- OCRは段階的移行対応: `/api/ocr` を優先し、失敗時にブラウザ内OCRへフォールバック
- Pages Functions `/api/ocr` は外部OCR連携プロキシ実装（`OCR_BACKEND_URL` 必須）
- 枠は常時編集可能、フォーカス中は青系の選択表示
- コピー時はボタン表示を一時的に「コピーしました」へ変更

## Cloudflare Pages Functions 環境変数
- `OCR_BACKEND_URL`: OCRバックエンドのPOSTエンドポイント（必須）
- `OCR_BACKEND_TOKEN`: バックエンド認証トークン（任意、Bearerとして送信）
