# OCR Keeper

ダークモードUIのブラウザOCRツールです。複数画像から文字列抽出し、固定枠で保存・編集・コピーできます。

## 仕様
- 最大20ページ / 1ページ10枠
- 写真は追加選択でキュー末尾に追記（上限200枚）
- 検索条件は常時表示（先頭一致 / 末尾一致 + 複数文字条件）
- OCR進行率バーで読み込み進捗を表示
- OCRは段階的移行対応: まず `/api/ocr` を試し、失敗時にブラウザ内OCRへフォールバック
- OCR失敗時は「読み取れませんでした」で枠消費
- 枠は常時編集可能、フォーカス中は枠色で選択状態を表示
- コピー時はアイコンが一時的にチェック表示に変化
